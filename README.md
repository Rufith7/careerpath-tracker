A full-stack career guidance platform with authentication, quizzes, resume builder, and AI-based recommendations.

##  Features
-  Dashboard with multiple domains
-  Quiz system with progress tracking
-  Resume builder
-  Career recommendations

##  Tech Stack
- Frontend: React
- Backend: Node.js + Express

##  Run Locally

### Backend
```bash
npm install
node stable-server.js
```
## Frontend 
cd client
npm install
npm start
